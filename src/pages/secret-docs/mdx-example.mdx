# MDX Example

This is a simple example of an MDX file in action. MDX lets you use JSX in your markdown content.

## Basic Components

Here's a simple JSX component:

export function Highlight({ children }) {
  return (
    <div style={{ 
      padding: '1rem',
      marginBottom: '1rem',
      border: '1px solid #ccc',
      borderRadius: '0.5rem',
      backgroundColor: '#f8f9fa' 
    }}>
      {children}
    </div>
  );
}

<Highlight>
  This content is wrapped in a styled container thanks to JSX!
</Highlight>

## Regular Markdown Still Works

- You can still use **bold** and *italic* text
- Lists work normally
- Code blocks work too:

```js
// This is a JavaScript code block
function greet(name) {
  return `Hello, ${name}!`;
}
```

## Mixing Markdown and JSX

You can mix Markdown and JSX freely:

<div style={{ color: 'blue' }}>
  ### This heading is blue
  
  This text is also blue, and it's inside a div.
</div>

## Using Props

You can even pass props from the frontmatter to your components!

export const Greeting = ({ name }) => <div>Hello, {name}!</div>;

<Greeting name="Astro User" />
