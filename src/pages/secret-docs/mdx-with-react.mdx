## Static JSX

You can also define simpler JSX elements directly in your MDX:

export function ColoredBox({ color, children }) {
  return (
    <div style={{ 
      padding: '1rem',
      marginBottom: '1rem', 
      backgroundColor: color,
      color: 'white',
      borderRadius: '0.5rem'
    }}>
      {children}
    </div>
  );
}

## Using Markdown with JSX

You can seamlessly blend Markdown syntax with JSX components:

<div style={{ display: 'flex', gap: '1rem', flexWrap: 'wrap' }}>
  <div style={{ flex: '1 1 200px', border: '1px solid #ddd', padding: '1rem', borderRadius: '0.5rem' }}>
    ### Left Column
    
    - This is a list in **Markdown**
    - Inside a JSX div
    - Pretty neat!
  </div>
  
  <div style={{ flex: '1 1 200px', border: '1px solid #ddd', padding: '1rem', borderRadius: '0.5rem' }}>
    ### Right Column
    
    ```js
    // Code blocks work too
    function greet() {
      return "Hello from MDX!";
    }
    ```
  </div>
</div>

## Benefits of MDX

1. **Component Reuse** - Import and reuse React components
2. **Interactive Content** - Add dynamic elements to your documentation
3. **Familiar Syntax** - Use the Markdown you already know
4. **Flexible Layouts** - Combine the simplicity of Markdown with the power of JSX
