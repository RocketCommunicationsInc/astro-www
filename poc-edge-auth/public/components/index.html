Available Components

Button Component

A versatile button component with multiple variants.

<astro-button variant="primary">Click Me</astro-button>

Basic Properties:

  • variant: primary, secondary, danger
  • size: small, medium, large

Card Component

A container component for displaying grouped content.

<astro-card title="My Card">Content here</astro-card>

Basic Properties:

  • title: Card header text
  • bordered: Show border (true/false)

Alert Component

Display important messages to users.

<astro-alert type="info">Information message</astro-alert>

Basic Properties:

  • type: info, success, warning, error
  • dismissible: Allow closing (true/false)
